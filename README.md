# 📚 우리아이 공부 스케줄러 & 데일리 To-Do (V2)

온 가족이 공유하고 함께 체크하는 자녀 공부 스케줄표 및 데일리 To-Do 시스템입니다.
본 버전은 기존 시놀로지 NAS 전용 PHP 버전을 전면 개편하여, **GitHub 저장소와 Vercel 배포 환경**에 완벽하게 최적화된 Modern Web App으로 새롭게 구현되었습니다.

---

## ✨ 주요 기능 및 특징

### 👧 아이 공부방 (`index.html`)
- **접속 잠금 PIN** — 외부 사용자 접속 차단을 위한 4자리 비밀번호 (기본값: `0000` - 변경 및 비활성화 가능)
- **자녀 프로필 선택** — 넷플릭스 스타일의 귀여운 프로필 카드 UI 지원
- **오늘의 할 일 목록** — 오늘 요일에 매칭되는 과목들이 자동으로 생성되고, 완료 시 폭죽 이펙트와 함께 완료 시간 기록
- **공부 달성률 그래프** — 완료 상황에 따라 실시간으로 변화하는 프로그레스 바 제공
- **연속 달성 스트릭** — 일정을 완료할 때마다 늘어나는 불꽃 스트릭 숫자로 자녀의 학습 동기 부여
- **주간 도장판 캘린더** — 월요일부터 일요일까지 요일별 완수 상황을 도장(성공/진행중)으로 보여줌

### 👩 부모 관리 모드 (`admin.html`)
- **관리자 PIN 인증** — 부모 전용 관리자 암호 4자리로 진입 통제 (기본값: `1234`)
- **주간 스케줄 플래너** — 요일별 학습 과목 추가, 수정, 삭제 및 시간 입력 가능
- **멀티 데이 일괄 등록** — 과목 추가 시 여러 요일을 함께 체크하여 동시에 다중 등록 가능
- **스케줄 요일 복사** — 하루의 전체 스케줄을 통째로 다른 요일로 손쉽게 복사/붙여넣기
- **오늘 중 To-Do 추가** — 정기 일정이 아닌 오늘 하루만 해야 하는 돌발 숙제 추가 기능
- **상세 히스토리 타임라인** — 자녀별 날짜별 완료 비율 및 각각의 완료 시간 추적
- **자녀 관리 및 격려 문구 편집** — 프로필 추가/삭제 및 아이 공부방에 뜨는 응원 메시지 실시간 등록
- **데이터 백업 및 복원** — 전체 데이터셋을 하나의 JSON 파일로 다운로드 및 업로드 복원

## 🛠 시스템 아키텍처 및 데이터 동기화

1. **로컬 개발 모드 (`http://localhost:3000`)**
   - Express 웹 서버 (`server.js`)가 로컬에서 구동되며, 파일 시스템의 `data.json`에 일정을 보관하고 수정합니다.
2. **프로덕션 배포 모드 (Vercel)**
   - Vercel Serverless Function (`api/data.js`)이 작동합니다.
   - **Vercel KV (Redis)** 데이터베이스가 연동되어 있다면 실시간 서버 클라우드 데이터에 저장되어 가족들 기기 간 실시간 동기화가 이루어집니다.
   - KV가 연결되지 않은 임시 상태라면 기본 `data.json`을 가져와 사용합니다.
3. **오프라인 / 단독 구동 모드 (`file://` 프로토콜)**
   - HTML 파일을 로컬 컴퓨터에서 브라우저로 더블 클릭하여 직접 실행하는 경우, 백엔드 서버 없이 브라우저 내의 **LocalStorage**를 사용해 구동되므로 완벽한 오프라인 단독 사용이 가능합니다.

## 🚀 로컬 실행 방법

1. 의존성 설치:
   ```bash
   npm install
   ```
2. 로컬 개발 서버 실행:
   ```bash
   npm run dev
   ```
3. 웹 브라우저에서 아래 주소로 접속합니다:
   - 공부방: `http://localhost:3000`
   - 부모 설정: `http://localhost:3000/admin.html` (또는 공부방의 부모 아이콘 클릭)

## ☁️ Vercel 서버리스 배포 및 실시간 데이터 설정 방법

가족들이 서로 다른 스마트폰, 태블릿, PC에서 데이터를 실시간으로 동기화하려면 GitHub에 업로드한 뒤 Vercel에 배포하고 KV 저장소를 무료로 연결하면 됩니다.

1. **GitHub 업로드**:
   ```bash
   git remote add origin https://github.com/lsmsonic/scheduler.git
   git push -u origin master
   ```
2. **Vercel 프로젝트 연동**:
   - [vercel.com](https://vercel.com)에 로그인 후 **Add New... > Project** 선택
   - 방금 올린 `scheduler` 저장소를 가져와(Import) 추가 작업 없이 그대로 **Deploy** 합니다.
3. **실시간 데이터베이스(Vercel KV) 연결**:
   - Vercel 대시보드에서 생성된 프로젝트로 들어갑니다.
   - **Storage** 탭을 클릭하고 **Create Database**에서 **KV (Redis)**를 만듭니다.
   - 데이터베이스 생성 후 **Connect to Project**를 눌러 해당 웹 앱 프로젝트에 연결합니다.
   - 이제 서버리스 함수가 재시작되어 실시간 클라우드 DB 연동이 자동으로 개시됩니다! 🎉
